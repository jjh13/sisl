fftwalloc.hpp

An allocator that wraps FFTW Malloc, for properly aligned memory allocations.

Taken mostly from "The Mallocator" http://blogs.msdn.com/b/vcblog/archive/2008/08/28/the-mallocator.aspx With a few very minor changes.

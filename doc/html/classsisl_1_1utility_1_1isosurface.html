March isosurface.

Runs marching cubes (with corrected ambiguities) over f. It starts at origin, and marches a prismic volume with the dimensions in extent. If the dx,dy,dz are provided, then they will be used as the normal for the mesh, if not, then this function will attempt to use the grad member of the f. If that doesn't work, it'll use a divided difference to approximate the shading. The step size for this can be set via set_shading_stepsize.

Parameters
fthe input function
isoValueThe iso value to extract
stepSizeThe size of steps, or the length of the edges of the cubes
originWhere the marching begins in space
extentThe dimensions of the volume to march over
flip_normalWhether to flip the output normal.
dx,dy,dzIf specified, these are functions to use for the dx dy and dz gradient components.
